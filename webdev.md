## 💡 How I Study Web Dev

1. **Watch + Code along** — never just watch passively
2. **Push every project to GitHub** — no exceptions
3. **Break things intentionally** — best way to learn
4. **Build one extra small project** per major topic — personalizes learning
5. **Write a short note** on what I learned — helps retention
